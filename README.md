# Sistema S&OP: Análisis del Mercado Automotriz Nacional 🚗📊

Este repositorio contiene la arquitectura de datos y los scripts utilizados para desarrollar un modelo integral de Ventas y Operaciones (S&OP) enfocado en el sector automotriz en México.

## 🎯 Objetivo del Proyecto
Transformar datos macroeconómicos crudos en un flujo automatizado que permita conectar la demanda, inventario, producción y exportaciones para la toma de decisiones estratégicas a nivel Dirección General.

## 🛠️ Stack Tecnológico
* **Extracción y Transformación (ETL):** Python (Pandas)
* **Almacenamiento y Modelado:** SQLite / SQL Server
* **Visualización:** Power BI (DAX avanzado)

## 🏗️ Arquitectura de Datos
1. **Extracción:** Consumo de datos históricos y registros desde el portal del INEGI.
2. **Transformación:** Limpieza y estructuración de datos utilizando scripts de Python.
3. **Carga:** Modelado relacional en base de datos SQL.
4. **Visualización:** Despliegue en Power BI del "MDMA Dashboard".

*Puedes visualizar el diagrama de flujo detallado en el archivo PDF adjunto en este repositorio.*

## 📈 Impacto Proyectado (Business Case)
* Visibilidad 360° en 3 ejes clave: Ventas, Producción y Exportaciones.
